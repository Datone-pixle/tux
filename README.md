# Surgeon Tux

###### A jekyll theme for terminal lovers.


#### Requirements
```bash
$ gem install jekyll
$ gem install jekyll-paginate
```

#### Usage
To run :
```bash
$ jekyll serve
```
And log from your browser, http://localhost:4000 or http://127.0.0.1:4000
